# Iconography

SSH Keys Manager uses [Feather Icons React](https://github.com/feathericons/react-feather) for consistent, clean, and minimalist iconography throughout the application. Feather Icons is an open source collection of simply beautiful icons designed at 24x24 with an emphasis on consistency and simplicity.

## Installation

```bash
# Using npm
npm install react-feather

# Using yarn
yarn add react-feather
```

## Usage

To use Feather icons in your components:

```jsx
// Import the icons you need
import { Eye, Trash2, Plus } from 'react-feather';

// Use them in your JSX
function MyComponent() {
  return (
    <div>
      <Eye size={18} color="#333" /> View details
      <Trash2 size={18} color="red" /> Delete item
      <Plus size={18} /> Add new
    </div>
  );
}
```

## Best Practices

When using icons in the SSH Keys Manager application:

1. **Consistency**: Use the same icon for the same action across the application
2. **Size**: Maintain consistent sizing - typically 18px for inline icons and 24px for standalone icons
3. **Color**: Use colors that align with the application's color system
4. **Accessibility**: Always pair icons with text labels or aria-labels for screen readers
5. **Spacing**: Allow sufficient spacing around icons for clear visual distinction
